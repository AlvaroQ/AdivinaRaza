# iosApp

iOS host for the AdivinaRaza Compose Multiplatform app. The Xcode
project is generated from `project.yml` (xcodegen) and integrates the
KMP `AdivinaRazaShared.framework` via a build phase + CocoaPods for the
Firebase iOS / Google Mobile Ads SDKs that gitlive auto-links against.

## First-time setup (after a fresh clone)

Pre-requisites:
- Xcode 16.4 or newer
- `brew install xcodegen cocoapods`

```bash
cd iosApp
xcodegen generate     # produces iosApp.xcodeproj from project.yml
pod install           # produces iosApp.xcworkspace + Pods/
```

> **Always open `iosApp.xcworkspace`**, never the `.xcodeproj` directly,
> otherwise the Pods linkage breaks.

## Build from CLI

```bash
cd iosApp
xcodebuild \
  -workspace iosApp.xcworkspace \
  -scheme iosApp \
  -configuration Debug \
  -sdk iphonesimulator \
  ONLY_ACTIVE_ARCH=YES ARCHS=arm64 \
  build
```

Each build re-runs the `Build Kotlin Multiplatform Framework` phase,
which calls `:app:linkDebugFrameworkIosSimulatorArm64` so any change to
the shared Kotlin code is picked up automatically.

## What is checked into git

| Path | Tracked? | Why |
|------|----------|-----|
| `project.yml` | ✅ | Source of truth for `xcodegen generate` |
| `Podfile` | ✅ | Source of truth for `pod install` |
| `Podfile.lock` | ✅ | Reproducible Pods versions |
| `iosApp/iOSApp.swift`, `iosApp/Assets.xcassets/...` | ✅ | App source |
| `iosApp.xcodeproj/`, `iosApp.xcworkspace/`, `Pods/` | ❌ | Regenerated |
| `iosApp/Info.plist` | ❌ | Generated by xcodegen from `project.yml` |
| `GoogleService-Info.plist` | ❌ | Per-developer Firebase credentials |

## Adding `GoogleService-Info.plist`

Download from Firebase Console → Project Settings → your iOS app, drop
into `iosApp/iosApp/`, then:

```bash
xcodegen generate     # picks it up automatically as a resource
```
